# 운동 기록 (Daily Workout Log)

하루 단위로 운동과 갯수를 기록하는 웹페이지. 메모장에 적던 방식을 그대로 웹으로 옮겼습니다.

## 사용법

`index.html` 파일을 브라우저에서 열면 됩니다. 별도 설치·서버가 필요 없습니다.

## 기능

- **날짜별 기록** — 이전/다음 날 이동, 날짜 선택 (기본: 오늘)
- **부위별 분류** — 복근 / 등 / 가슴 등, 운동 옆에 갯수 입력
- **완료 표시** — 갯수를 입력한 운동은 초록색으로 표시
- **요약** — 오늘 한 운동 수 / 총 갯수 / 연속 기록일
- **편집 모드** — 운동·부위 추가·수정·삭제
- **지난 기록** — 기록한 날 목록 보기
- **백업** — JSON 내보내기 / 불러오기

## 데이터 저장

기록은 브라우저의 `localStorage`에 저장됩니다. 다른 기기와 공유하려면
`더보기 → 백업 내보내기`로 파일을 옮기세요.
